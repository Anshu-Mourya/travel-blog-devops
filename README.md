# 🌍 Travel Blog DevOps Project

A modern travel dashboard application built using frontend technologies and DevOps tools.

---

# 🚀 Project Overview

This project is a professional travel dashboard UI designed with:

- Modern pastel dashboard layout
- Responsive mobile-first design
- Interactive cards and animations
- Sidebar navigation
- Travel destination showcase

The project will later be extended with:

- Docker
- Kubernetes
- Helm
- Jenkins
- GitHub Actions
- Monitoring
- CI/CD pipelines

---

# 🛠️ Technologies Used
